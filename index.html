Elements HTML
# Element Name Decription
1 <h1> Heading 1 Creates a top-level heading that is used for major titles and is displayed with a larger font size than normal text.
2 <p> Paragraph Creates a block of text and is used to divide content into separate text blocks.
3 <a> Anchor Creates a link to another web page or a section within the same page.
4 <img> Image Inserts an image into the web page and is used to display graphics, illustrations, and photographs.
5 <ul> Unordered List Creates an unordered list of items and is used to organize items in a list without any specific order
6 <ol> Ordered List Creates an ordered list of items and is used to organize items in a list with a specific order
7 <div> Division elemnt Used to group together other HTML elements and apply styles or layout to them.
Elements CSS
# Name Decription
1 Class Allows you to apply styles to multiple HTML elements by grouping them together under a single class name.
2 ID Similar to a class, but is used to identify and style a single, unique HTML element.
3 Box model A concept that defines how CSS treats HTML elements as rectangular boxes with content, padding, borders, and margins.
4 Display Determines how an HTML element should be displayed, such as block, inline, or inline-block.
5 Position Determines how an element should be positioned within its container, such as static, relative, absolute, or fixed.
6 Flexbox A layout model that makes it easier to create flexible and responsive layouts with CSS, especially for complex and dynamic interfaces.
7 Grid A newer layout model that allows you to create complex and flexible layouts with rows and columns, similar to a spreadsheet.
